
📊 Git Repository Insights Tool

An Android application that provides detailed insights into any public GitHub repository. The app helps developers analyze repository statistics, contributors, commits, and project activity through an easy-to-use interface.

---

## 📖 Project Overview

Developers often visit multiple GitHub pages to understand a repository's activity and contributions. This project brings all the important repository information together in one Android application, making repository analysis faster and more convenient.

The application uses the GitHub REST API to fetch real-time repository data and presents it in a clean and organized dashboard.

---

## ✨ Features

- 🔐 User Login using Firebase Authentication
- 🔍 Search any public GitHub repository
- 📂 View repository details
- ⭐ Repository statistics (Stars, Forks, Issues, Watchers)
- 👥 Contributor insights
- 📝 Commit history
- 📈 Weekly repository activity
- 🛡️ Repository security information
- 🎨 Modern Material Design UI

---

## 🛠️ Technologies Used

- 💜 Kotlin
- 🤖 Android Studio
- 🎨 Jetpack Compose
- 🔥 Firebase Authentication
- 🐙 GitHub REST API
- 🌐 Retrofit
- 📦 Gson
- 🏗️ MVVM Architecture
- ⚙️ Gradle Kotlin DSL

---

## ⚡ How It Works

1. 🔑 User logs into the application.
2. 🔍 Enter the repository name.
3. 🌐 The app sends a request to the GitHub API.
4. 📥 Repository data is fetched.
5. 📊 The information is displayed through different analytics screens.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/thanushree-v010604/Git-Repository-Insights-Tool.git
```

2. 📂 Open the project in Android Studio.

3. 🔄 Sync Gradle.

4. 🔥 Add your Firebase configuration file (`google-services.json`).

5. ▶️ Run the application.
